# mobile-app-react-native

A high-quality mobile app built using React Native.

## Features

*   Customizable UI components
*   Robust navigation system
*   Advanced state management
*   Real-time data fetching
*   Offline support

## Dependencies

*   React Native: ^0.69.0
*   React: 17.0.2
*   Redux: ^8.0.2
*   Redux-Saga: ^1.1.3
*   Axios: ^0.21.4
*   Lodash: ^4.17.21

## Installation

```bash
npm install
```

## Usage

```bash
npm start
```
